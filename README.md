# Тест архетипов (match3)

Мини-приложение на TypeScript и чистом HTML/CSS: свайповый тест из 16 карточек с подсчётом архетипов, кнопками лайка/дизлайка и итоговым профилем.

## Запуск

```bash
npm install
npm run build
npm run start
```

- `npm run build` — собирает TypeScript в `dist/` через esbuild и копирует статику.
- `npm run start` — собирает и поднимает локальный превью-сервер на `http://localhost:3000` (порт из `serve`).
- Для быстрой разработки можно запустить `npm run dev` и открыть `src/index.html` с подключённым `main.js` из `dist` (режим watch).

После билда достаточно открыть `dist/index.html` в браузере — вся логика работает локально, сеть не требуется.

## Как играть

1. Пролистайте обучающее модальное окно (кнопка «Понятно» или `?`).
2. Смахивайте карточки вправо (нравится) или влево (не нравится). Можно использовать нижние кнопки или мышь.
3. Проходите все 16 карточек. Вверху отображается прогресс.
4. После последнего ответа появится итоговый профиль: главный архетип и проценты по топ‑5. В коде есть заготовка для отправки результатов на сервер.

## Расчёт

- У каждой карточки есть состав архетипов; лайк добавляет веса в копилки, дизлайк вычитает их с «мягким» коэффициентом 0.7.
- Экспозиция учитывается для нормализации; если все копилки не положительные, профиль строится по экспозиции.
- Данные карточек и описания архетипов лежат в `src/main.ts` и могут быть заменены на изображения/тексты дизайнера.

## Структура

```
src/
  index.html   — разметка приложения
  style.css    — стили и анимации свайпа
  main.ts      — логика свайпов, подсчёта и итогов
```
